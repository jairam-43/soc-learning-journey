# SOC Roles and Responsibilities

A Security Operations Center monitors, detects, investigates, and responds to security events inside an organization.

## Tier 1 – Alert Analyst

- Monitors SIEM dashboards
- Triages alerts
- Escalates incidents
- Documents findings

## Tier 2 – Incident Responder

- Investigates escalated alerts
- Performs log analysis and correlation
- Determines root cause and impact
- Contains ongoing security incidents

## Tier 3 – Threat Hunter

- Proactively searches for threats
- Builds detection rules
- Analyzes attacker behavior and TTPs
- Works with threat intelligence feeds

## SOC Manager

- Oversees SOC operations
- Defines workflows and SLAs
- Coordinates with other teams (IR, IT, Red Team)

## Key Goal

Detect attacks early, respond quickly, and maintain visibility across endpoints, networks, and cloud systems.
