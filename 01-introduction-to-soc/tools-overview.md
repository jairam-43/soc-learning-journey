# SOC Tools Overview

SOC analysts use a mix of monitoring, detection, and investigation tools.

## SIEM

- Wazuh
- ELK Stack
- Splunk
- QRadar

Purpose: Collect logs, correlate events, trigger alerts.

## Endpoint Detection

- Wazuh agents
- CrowdStrike Falcon
- Microsoft Defender for Endpoint

Purpose: Monitor processes, network activity, file changes.

## Network Monitoring

- Suricata
- Snort
- Zeek
- Wireshark

Purpose: Detect malicious traffic, signatures, anomalies.

## Threat Intelligence

- AbuseIPDB
- AlienVault OTX
- VirusTotal
- MITRE ATT&CK

Purpose: Identify threat actors, TTPs, and IOCs.
