# Common Ports Every SOC Analyst Must Know

## Web
- 80 → HTTP
- 443 → HTTPS

## Remote Access
- 22 → SSH
- 3389 → RDP

## File Transfer
- 20/21 → FTP
- 69 → TFTP
- 445 → SMB

## Email
- 25 → SMTP
- 110 → POP3
- 143 → IMAP

## DNS
- 53 → DNS

Knowing ports helps you quickly identify suspicious traffic.
