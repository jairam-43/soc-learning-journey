# Sysmon Basics for SOC

Sysmon gives deeper visibility into system activity.

## Key Events

- **Event 1** → Process creation
- **Event 3** → Network connections
- **Event 7** → DLL loading
- **Event 11** → File creation
- **Event 13** → Registry modification

## Why It Matters

Sysmon helps detect:
- Persistence techniques
- Malware execution
- Lateral movement
- Suspicious network traffic
