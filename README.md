# SOC Learning Journey

This repository documents my complete learning path toward becoming a SOC Analyst. It includes structured notes, hands-on labs, log analysis work, detection experiments, and threat-hunting practice. Everything in this repo is based on real daily learning and practical exercises.

## Contents

### 1. Introduction to SOC
Roles, responsibilities, tier levels, common tools, and basic MITRE ATT&CK understanding.

### 2. Networking Fundamentals
TCP/IP, ports, protocols, OSI layers, packet analysis, and Nmap basics.

### 3. Linux for SOC
File system, permissions, processes, logs, and essential commands from a security perspective.

### 4. Windows for SOC
Event Viewer, Sysmon, registry, Windows log types, and common artifacts.

### 5. Log Analysis
Syslog, authentication logs, event types, and sample real-world analysis exercises.

### 6. Alerts and Incidents
Alert severity, incident lifecycle, and sample incident write-ups.

### 7. SIEM Labs
Wazuh, ELK, rule creation, dashboarding, and detection examples.

### 8. Threat Detection
Rule writing, suspicious behavior patterns, and basic threat-hunting scenarios.

## How to Use This Repo

Each folder contains focused notes and hands-on tasks. I continuously update this repo as I progress through my SOC roadmap.
